# 🖱️ Cursor Control with Hand Gestures

Control your computer cursor using just your hand ✋
This project uses Computer Vision to track hand movements and perform mouse actions like moving, clicking, and scrolling — all in real time!

---

## 🚀 Features

* Real-time hand tracking using webcam
* Control mouse movement with hand gestures
* Left click using finger gesture
* Right click / scrolling (customizable)
* Smooth and responsive cursor control
* Beginner-friendly project

---

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* NumPy

## ⚙️ Installation

1. Clone the repository:

git clone https://github.com/your-username/cursor-control-hand-gesture.git
cd cursor-control-hand-gesture

2. Install required libraries:

pip install -r requirements.txt

---

## ▶️ How to Run

python main.py

Make sure your webcam is ON

---

## ✋ Gesture Controls

Index Finger → Move cursor
Index + Middle Finger → Left click
Pinch Gesture → Drag / Scroll
Fist → Pause movement
